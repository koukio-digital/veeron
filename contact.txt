# Usage notes:
# 1) In the target Google Sheet set the first row headers to:
#    Timestamp	Name	Mobile	Email	Company	Service	Message
# 2) In Apps Script, deploy as "Web App" (Execute as: Me, Who has access: Anyone).
# 3) Use the provided Web App URL as GAS_URL in `contact.html` for submissions.
# 4) Example client POST payload (JSON):
#    {
#      "Name":"Vikram Mehta",
#      "Mobile":"9876543210",
#      "Email":"vikram@company.com",
#      "Company":"Zenith Global Enterprises",
#      "Service":"App Development",
#      "Message":"Interested in a cross-platform mobile app."
#    }

# 5) For local testing, append rows to this file manually or use the deployed Web App.
